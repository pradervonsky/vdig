# VBIG: VLM for BI-Dashboard Insight Generation  

---

**VBIG** is a prototype system that explores how a **small Vision-Language Model (2B parameters)** can augment business intelligence workflows by generating **concise, human-readable insights from BI dashboards**.

The system combines:
- **Vision-based insight generation** (self-hosted VLM)
- **Agentic orchestration** (self-hosted n8n)
- **Supabase storage and database** (dashboards + historical insights)
- **Human-in-the-loop step** (to approve AI-generated insights via Telegram)
- **Daily dashboard simulation**

VBIG serves as the applied component of a broader goal: 
**Embedding lightweight AI assistants into enterprise BI systems to improve clarity, speed, and decision support.**
